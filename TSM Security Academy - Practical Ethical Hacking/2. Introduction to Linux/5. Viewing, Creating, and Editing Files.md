## Echo

- Can be used to create and write a File
- echo "hey" > hey.txt
- echo "hey again" > hey.txt
    - Overwrites the File
- echo "hey again" >> hey.txt
    - Append to the end of a File

## Cat

- Print the data inside a File

## Touch

- Create a File
    - touch newfile.txt

## Nano

- Terminal Text Editor
- nano newfile.txt
    - Opens a text editor

## Gedit

- Graphical interface to edit files
